# Cool Galaxy Backend

A backend service for the Cool Galaxy project.

## Overview

This is the backend repository for the Cool Galaxy application, built with modern web technologies.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```

## Development

Currently on the `dev` branch. Push changes to `dev` before merging to `main`.

```bash
npm run dev
```
